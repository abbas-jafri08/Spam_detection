# Spam Detector

Backend: Flask + scikit-learn (model files in backend/)
Frontend: React (Vite) in frontend/

## Quick start
1. Backend:
   - cd backend
   - python -m venv venv
   - venv\Scripts\activate  # Windows
   - pip install -r requirements.txt
   - python app.py

2. Frontend:
   - cd frontend
   - npm install
   - npm run dev
